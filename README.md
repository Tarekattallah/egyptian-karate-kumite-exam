# امتحان الكومتيه | الاتحاد المصري للكاراتيه

<img src="assets/images/logo.png" alt="EMKA Exam UI Logo" width="80">

تطبيق ويب تفاعلي باللغة العربية لإجراء امتحانات الكاراتيه (الكومتيه) تحت إشراف الاتحاد المصري للكاراتيه. يقدّم تجربة مستخدم متكاملة للطلاب والاعبين مع واجهة عصرية، وتتبع فوري للتقدم، وآلية تقييم تلقائية.

---

## 🛠️ التقنيات المستخدمة

| التقنية | التفاصيل |
|---|---|
| **HTML5** | هيكل الصفحات والدلالات |
| **CSS3** | Flexbox، Animations، Media Queries |
| **JavaScript (ES6+)** | Vanilla JS — Async/Await، DOM manipulation |
| **Font Awesome 6** | الأيقونات |
| **Google Fonts** | خط Tajawal |

**لا يوجد أي Backend أو frameworks** — ملف مستقل يعمل مباشرة في المتصفح.

## 🎯 الميزات

- **عداد زمني 60 دقيقة** مع تنبيه تلقائي عند انتهاء الوقت
- **لوحة أسئلة تفاعلية** للتنقل السريع بين الأسئلة
- **شريط تقدم متحرك** يوضح نسبة الإنجاز (%)
- **إحصائيات حية** (صحيحة / خاطئة / متبقية) تتحدث تلقائياً
- **نظام إجابة فوري** مع تغذية راجعة فورية تشمل:
  - علامة الإجابة الصحيحة بالعربية
  - شرح السبب
  - **المرجع** (المادة + رقم الصفحة)
- **لوحة ألوان ذكية** لكل سؤال (أخضر/أحمر/رمادي)
- **تقييم تلقائي** في نهاية الامتحان مع نسبة نجاح
- **زر إعادة الامتحان** بدون الحاجة لإعادة تحميل الصفحة
- **تصميم متجاوب** — يعمل على أحجام الشاشات المختلفة
- **دعم كامل للغة العربية RTL**

## 🚀 طريقة الاستخدام

1. افتح ملف `index.html` في أي متصفح حديث
2. انتظر تحميل الأسئلة تلقائياً من `data/questions.json`
3. ابدأ الامتحان واختر إجابة لكل سؤال (صحيح / خطأ)
4. تنقل بين الأسئلة عبر الأزرار أو لوحة الأسئلة السريعة
5. اضغط **"إنهاء الامتحان"** أو انتظر حتى ينتهي الوقت

## 🗂️ تنسيق ملف الأسئلة `data/questions.json`

```json
{
  "id": 1,
  "arabic": "نص السؤال بالعربية",
  "english": "Question in English",
  "type": "boolean",
  "options": ["صحيح", "خطأ"],
  "correctAnswer": false,
  "correctAnswerArabic": "خطأ",
  "explanation": "السبب أو الشرح بالإجابة",
  "reference": {
    "article": "المادة 5",
    "page": 12
  }
}
```

| الحقل | الوصف |
|---|---|
| `id` | رقم السؤال الفريد |
| `arabic` | نص السؤال بالعربية |
| `english` | نص السؤال بالإنجليزية |
| `type` | نوع السؤال (`boolean`) |
| `options` | الخيارات المتاحة |
| `correctAnswer` | الإجابة الصحيحة (boolean) |
| `correctAnswerArabic` | نص الإجابة الصحيحة بالعربية |
| `explanation` | الشرح التفصيلي |
| `reference.article` | اسم المادة/القاعدة |
| `reference.page` | رقم الصفحة في المصدر |

## ⚙️ سلوك النظام

| الحالة | التفاصيل |
|---|---|
| **اختيار إجابة** | يُغلق تلقائياً — لا يمكن تغيير الإجابة بعد الاختيار |
| **إجابة خاطئة** | يظهر التنبيه فوراً مع الشرح والمرجع |
| **إجابة صحيحة** | يظهر مؤشر نجاح مع التحديث التلقائي للإحصائيات |
| **انتهاء الوقت** | تنبيه بصري + إنهاء تلقائي للامتحان |
| **نسبة النجاح** | 60% فأعلى = نجاح |
